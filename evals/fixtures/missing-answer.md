## Executive Summary

Kubernetes is a container orchestration platform originally developed by Google. It manages containerized workloads across clusters of machines, handling scheduling, scaling, and self-healing.

## Key Findings

1. **Kubernetes was open-sourced by Google in 2014** — Confidence: Established
   Based on Google's internal Borg system. (Source: #1)

2. **Kubernetes uses declarative configuration via YAML manifests** — Confidence: Established
   Users define desired state; the control plane reconciles actual state to match. (Source: #1, #2)
